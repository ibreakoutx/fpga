Downloading and Running encrypt_app

1. Bringup EDK 9.1.02 (with service pack 2)
2. Hyperterminal connection to COM port, 115K, 8 bit data, 1 bit stop, no parity3. Download bit image to board
4. Bringup XMD
5. In XMD window
   5.1 cd encrypt_app
   5.2 dow executable.elf
   5.3 To restart app: stop. goto 5.2
